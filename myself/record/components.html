# rem布局 公式

  1. 原理根据改变根元素的font-size 来计算页面大小 (屏幕的宽度 / 设计稿的大小 * 一个基数(例如100) = fontsize) 1920/1920*100 = 100, fontSize= 100px 也就是 1rem = 100px 1px = 1/100 rem 10px = (1/100) *10
  2. px 转rem: px / 基数(例如100)
  3. rem 转 px: rem值 * 根元素
